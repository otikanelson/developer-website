API Reference

RESTful API

Our comprehensive REST API provides access to all SKG services and features.

GET /api/v1/games
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json

Popular Endpoints

  • GET /api/v1/games - List all games
  • POST /api/v1/games - Create new game
  • GET /api/v1/users/{id} - Get user profile
  • POST /api/v1/analytics - Submit analytics data
  • GET /api/v1/leaderboards - Fetch leaderboards
